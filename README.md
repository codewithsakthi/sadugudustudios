# Sadugudu Studios

A clean, modern, framework-free web project built with pure HTML5, Vanilla CSS, and JavaScript.

## 🚀 Project Structure

```
sadugudustudios/
├── index.html     # Main HTML page
├── styles.css     # Design system, CSS variables & responsive layout
└── script.js      # Client-side interactive logic & theme switching
```

## 🛠️ Features

- **Framework-free**: Lightweight & fast loading
- **Dark & Light Mode**: Built-in theme switcher with local storage persistence
- **Responsive Design**: Looks great on desktop, tablet, and mobile screens
- **Interactive JavaScript**: Simple counter component and theme toggle demo
